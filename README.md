# Campaign Userscripts

Tampermonkey scripts for List Manager and Rocket.

## Install

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. Click the script link below
3. Auto-updates when new versions are pushed

**[listmanager.user.js](https://raw.githubusercontent.com/choujar/campaign-userscripts/main/listmanager.user.js)** - SMS templates, copy buttons, electorate lookup, etc.
